Features

Everything you need.
Nothing you don't.

Built for power users who live in the terminal — fast, scriptable, and beautiful out of the box.

01 /
🎨

Beautiful TUI

Rich terminal interface with multiple themes — Dark, Light, Midnight, and Matrix. ASCII art cover art, sharp animations.

02 /
🖥️

Immersive Mode (Windows)

Fullscreen visual experience with audio visualizers and disco effects. Built with Win32 + kernel32.

03 /
🔍

Powerful Search

Find songs, albums, artists, and playlists instantly across the full YouTube Music catalog.

04 /
📋

Queue Management

Build and manage your playback queue with ease. Reorder, shuffle, repeat — all from the keyboard.

05 /
❤️

Favorites

Mark tracks with f and access your collection with Shift+F. Persistent across sessions.

06 /
🔀

Shuffle & Repeat

Multiple playback modes to suit your listening style — One, All, Shuffle, or no repeat.

07 /
🔌

Plugin System

Extend functionality with plugins like adblock, lyrics, Discord Rich Presence, and tray-icon. Build your own.

08 /
📻

Internet Radio

Browse and play Polish and international radio stations offline. 15 Polish stations built-in (RMF FM, Radio ZET, Polskie Radio, Eska, Antyradio i więcej).

09 /
⌨️

Keyboard-Driven

Efficient vim-style navigation for power users. Configurable shortcuts, mousetrap integration, full keymap.

10 /
🚀

Raycast Extension

Search, play, and control playback directly from Raycast — no terminal needed for quick actions. Built-in overlay and web server backend.

Quick start

Three lines.
You're playing music.

Install via your favourite package manager. No config required to get started.

# Install globally $ npm install -g @netflyapp/youtube-music-cli # Run $ youtube-music-cli $ ymc # Dependencies $ brew install mpv yt-dlp # macOS $ sudo apt install mpv yt-dlp # Debian/Ubuntu $ winget install mpv yt-dlp # Windows
# Install globally $ bun install -g @netflyapp/youtube-music-cli # Run $ bunx youtube-music-cli # Dependencies $ brew install mpv yt-dlp # macOS $ sudo apt install mpv yt-dlp # Linux
# Install globally $ pnpm add -g @netflyapp/youtube-music-cli # Run $ youtube-music-cli $ ymc # Dependencies $ brew install mpv yt-dlp # macOS $ winget install mpv yt-dlp # Windows
# macOS / Linuxbrew $ brew tap netflyapp/youtube-music-cli https://github.com/netflyapp/youtube-music-cli.git $ brew install youtube-music-cli # Run $ youtube-music-cli $ ymc

Requirements

Need three things.

Node.js plus two well-known open-source media tools.

🟢

Node.js 18+

Runtime for the CLI. Anything ≥ 18 works.

🎬

mpv

Lightweight media player — handles audio decoding.

⬇️

yt-dlp

YouTube audio extraction, the modern youtube-dl fork.
