Keyboard Shortcuts

Complete reference of all keyboard shortcuts in youtube-music-cli.

Global Shortcuts

These work from any screen.

| Key | Action |

-----------------------------

| ? | Show help screen |

/Open search

| p | Open plugins manager |

Shift+FOpen favorites view

| g | Show suggestions |

,Open settings

| q | Quit application |

| Esc | Go back / Close menu |

Playback Controls

| Key | Action |

| ----------------- | ----------------------------------- |

SpacePlay / Pause

| n | Next track |

→ (Right arrow)Next track

| b | Previous track |

| ← (Left arrow) | Previous track |

| Shift+→ | Seek forward 10 seconds |

| Shift+← | Seek backward 10 seconds |

| = | Volume up (+5%) |

-Volume down (-5%)

| s | Toggle shuffle |

| r | Cycle repeat mode (off → all → one) |

Navigation

| Key | Action |

| ----------- | --------------------------- |

| ↑ / k | Move selection up |

| ↓ / j | Move selection down |

EnterSelect / Confirm

| Esc | Go back |

| Tab | Next result (in search) |

| Shift+Tab | Previous result (in search) |

Radio

| Key | Action |

| ------------- | ------------------------------------- |

| R | Open radio station browser |

| Shift+X | Start/stop radio mode from current track |

| ↑ / ↓ | Navigate stations / countries |

| Enter | Select country / Play station |

| / | Search stations (in radio browser) |

| Esc | Go back |

Search

| Key | Action |

| --------- | -------------------------------------- |

/Focus search input

| Enter | Execute search |

EscClear search / Close

| [ | Show fewer results |

| Shift+D | Download selected song/artist/playlist |

Playlist

| Key | Action |

-----------------------------------
dRemove from playlist
cCreate new playlist

| D | Delete playlist |

| Shift+D | Download selected playlist |

Plugins Manager

| Key | Action |

---------------------------------------

| i | Install plugin |

eEnable/disable selected

| u | Update plugin |

| Tab | Switch between installed/available |

Settings Menu

| Key | Action |

| --------- | ---------------- |

↑ / ↓Navigate options

| Esc | Close settings |

Help Screen

| Key | Action |

-----------------
EscClose help

| q | Close help |

Tips

Vim-style Navigation

If you're familiar with vim, you can use:

  • j instead of ↓
  • k instead of ↑

Quick Access

  • Press / anywhere to quickly search
  • Press Space anywhere to play/pause
  • Press ? to see context-specific help

Volume Shortcuts

Note that volume uses = (equals) for up and - (minus) for down. This avoids issues with + requiring Shift on most keyboards.
